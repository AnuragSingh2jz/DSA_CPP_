In C++, function parameters can be passed by value or by reference.
 When a parameter is passed by value, a copy of the argument is made 
 and the function works with this copy. Any modifications made to the 
 parameter inside the function do not affect the original variable
  passed as an argument.

In the case of your input function:

cpp
Copy code
node *input(node *head, int value)
The head parameter is passed by value. 
This means that when you call the function and pass the head pointer,
 a copy of the head pointer is created inside the function.
  Any modifications to this copy, such as updating it to point 
  to a new node, only affect the local copy of head within the 
  function. The original head pointer outside the function remains
   unchanged.

To ensure that changes to head are reflected outside the function, you have a couple of options:

Pass head by reference:
You can modify the function signature to accept a reference to the head pointer:
cpp
Copy code
node *input(node *&head, int value)
By passing head by reference (node *&head), any changes 
made to head inside the function will directly affect the
 original head pointer outside the function.
Return the updated head pointer:
Instead of modifying the head pointer inside the function,
 you can return the updated head pointer and assign it to the head variable outside the function:
cpp
Copy code
node *head = nullptr;
// Call the function and assign the returned value to head
head = input(head, value);
This way, any changes made to head inside the function are
 captured by the return value and assigned to the head variable outside the function.
Both approaches ensure that changes made to head inside
 the function are reflected outside the function,
  allowing you to maintain consistency in the linked list.
